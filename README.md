# Luigi Cabrera Portfolio

A modern, high-performance personal portfolio built with **Next.js 14** and **Motion**. This project features a fluid user experience with custom animations, responsive design, and interactive elements.



## üöÄ Features

* **Next.js App Router:** Optimized routing and server-side rendering.

* **Interactive Background:** A "Liquid Ether" 3D background.
* **Responsive Modals:** Accessible, full-screen image viewing with backdrop blurring and smart aspect-ratio fitting.
* **Tailwind CSS:** Fully responsive UI with a focus on modern typography and dark-mode aesthetics.
* **TypeScript:** Type-safe development for better maintainability.
